# Design Tokens
This repository contains the design guidelines which all frontend applications and components in Umbrlla must follow.

## Font
For all copy text, the *Geomanist* font must be used.

| Name         | Value    
| ------------ | -----
| font-small   | 14px
| font-regular | 16px
| font-medium  | 18px
| font-large   | 24px
| font-xlarge  | 32px
